# Toy-Verse

**Toy-Verse** is an online local toy store platform where users can browse, explore, and request trial experiences for various toys. Built with React, Tailwind CSS, Firebase, and React Router, it provides a seamless and interactive shopping experience for both kids and parents.

---

---

## Key Features
- Browse a wide range of toys across categories like Building Blocks, Toy Cars, Arts & Crafts, and Dolls & Action Figures.
- View toy details including price, rating, available quantity, description, and seller information.
- Submit a "Try Now" request form to experience toys before purchasing.
- Responsive design with modern UI using Tailwind CSS.
- Interactive alerts using SweetAlert2 for notifications.
- Smooth navigation with React Router.
- Fully integrated with Firebase for backend functionality.

---

## Technologies Used
- **React** – Front-end library for building interactive UIs.
- **Tailwind CSS** – Utility-first CSS framework for rapid styling.
- **Firebase** – For authentication.
- **React Router** – Client-side routing for seamless navigation.
- **React Icons** – Icon library for UI elements.
- **SweetAlert2** – Beautiful alert messages for notifications and interactions.
